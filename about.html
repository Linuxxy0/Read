项目说明

这是一个为 GitHub Pages 设计的静态阅读站项目：无需后端、开箱即用、直接上传仓库即可部署。

已完成功能

首页独立展示、书架页单独加载、全站搜索、作品内搜索、动态标签统计、动态收录数、主题切换、作者页、分类页、阅读页字号调节，均已实现。

数据结构

项目通过 assets/data/books.json 提供作品、作者、标签、字数与节选内容。你后续只要更新 JSON，页面会自动联动。

部署方式

直接把整个项目目录推送到 GitHub 仓库，在 Settings → Pages 中选择 Deploy from a branch，使用根目录发布即可。

可继续扩展

后续可以加入完整正文、收藏、目录折叠、阅读进度、拼音检索或静态生成脚本。本次交付已保证核心功能可直接使用。
