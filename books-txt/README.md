# TXT 导入说明

把你的 `.txt` 书籍文件放到这个目录，然后在项目根目录运行：

```bash
./build-from-txt.sh
```

或在 Windows 双击：

```text
build-from-txt.bat
```

## 支持的 TXT 头部元数据（可选）

```text
标题: 浮生六记
作者: 沈复
年代: 清代
分类: 散文
标签: 散文, 生活, 清代, 小品
简介: 首页摘要
描述: 详情简介
精选: true
字数: 216000
```

元数据和正文之间空一行。

## 章节识别规则

脚本会自动把这些写法识别为章节标题：

- `# 第一章`
- `## 闲情记趣`
- `第一章 山居`
- `【湖心亭看雪】`

如果你的 TXT 没有章节标题，也可以直接导入，系统会自动生成为一个“正文”章节。
